# Recu Classic

Recu Classic — маленькая Win32-программа для восстановления удаленных файлов с флешек, SD-карт и образов `.img/.dd`. Никакой рекламы, красивеньких интерфейсов и установщиков на 300+ Мб)

Поддерживаются FAT16, FAT32 и exFAT. NTFS в разработке :)

![Recu Classic](docs/screenshots/recu-classic-main-ru.png)

## Что умеет прога

- искать удаленные файлы по записям файловой системы;
- искать файлы по сигнатурам, если записи уже нет, поддерживаются более 30 расширений;
- создавать образ носителя и сканировать его (безопасный режим);
- восстанавливать файлы с сохранением папок;
- показывать превью JPEG/PNG и метаданные EXIF;
- проверять найденные и восстановленные файлы;
- показывать шанс восстановления и причины;
- писать отчет о диске и лог восстановления;

## Сборка

```sh
make all
make test
make portable
```

Готовый архив появляется здесь:

```text
dist/RecuClassic-portable-win64.zip
```

## Ограничения

- NTFS пока нет
- Глубокое сканирование может находить не все файлы или мусор
- Если данные уже перезаписаны, то восстановление зачастую невозможно, смиритесь :)
- Для работы с дисками напрямую (без образа) требуются права админа
