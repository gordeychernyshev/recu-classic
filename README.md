# Recu Classic

Recu Classic — портативная программа под Windows для восстановления удаленных файлов с флешек, SD-карт и образов дисков `.img/.dd`.

Программа работает с FAT16, FAT32 и exFAT. Это обычные файловые системы для USB-флешек, карт памяти фотоаппаратов, старых SD-карт и некоторых внешних носителей. NTFS пока не поддерживается.

## Как это работает

Recu Classic читает исходный носитель в режиме только чтения. Записывать восстановленные файлы на тот же диск нельзя: так легко перезаписать данные, которые еще можно было восстановить.

Есть два режима поиска.

Быстрое сканирование смотрит удаленные записи файловой системы. Если запись сохранилась, у файла обычно остается имя, размер и путь к папке.

Глубокое сканирование проходит область данных и ищет файлы по сигнатурам. Так можно найти файлы без имени и папки, но будут ложные совпадения. Для таких результатов важны проверка файла, размер и оценка шанса восстановления.

Если носитель важный или ведет себя странно, лучше сначала создать образ, а потом сканировать уже его.

## Что есть сейчас

- FAT16, FAT32 и exFAT.
- Работа с физическими носителями и образами `.img/.dd`.
- Создание образа носителя.
- Быстрое и глубокое сканирование.
- Восстановление с сохранением исходных папок.
- Предпросмотр JPEG/PNG.
- EXIF-данные фотографий и поворот превью.
- Проверка найденных и восстановленных файлов.
- Оценка шанса восстановления с причинами.
- Поиск дублей и пересечений.
- Защита от восстановления на тот же диск.
- Отчеты и лог восстановления в CSV, JSON и LOG.
- Русский и английский интерфейс.
- Портативная сборка без установки.

## Готовая сборка

Архив для Windows x64 находится в GitHub Releases:

`Recu Classic v1.0.0-beta.1 portable win64`

Внутри архива:

```text
recu-classic.exe
recu-cli.exe
Start Recu Classic.cmd
config/settings.ini
logs/
reports/
recovered/
```

Настройки портативной сборки хранятся рядом с программой в `config/settings.ini`.

## Сборка из исходников

Нужен Windows и `w64devkit`, который лежит рядом с проектом в этой рабочей папке.

```sh
make all
make test
make portable
```

Основные файлы после сборки:

```text
bin/recu-classic.exe
bin/recu-cli.exe
dist/RecuClassic-portable-win64.zip
```

## Ограничения

- NTFS пока не поддерживается.
- Глубокое сканирование может находить мусор и ложные совпадения.
- Если данные уже перезаписаны, файл восстановится поврежденным или не откроется.
- Для чтения физических дисков Windows может потребовать запуск от имени администратора.
- Программа не заменяет профессиональную лабораторию, если носитель физически умирает.

## Правило нормального использования

Восстанавливайте только свои данные или данные, на восстановление которых у вас есть разрешение.
