# Recu Classic

Recu Classic — маленькая Win32-программа для восстановления удаленных файлов с флешек, SD-карт и образов `.img/.dd`.

Поддерживаются FAT16, FAT32 и exFAT. Это частый формат для флешек, карт памяти фотоаппаратов и старых носителей.

![Recu Classic](docs/screenshots/recu-classic-main-ru.png)

## Что умеет

- искать удаленные файлы по записям файловой системы;
- искать файлы по сигнатурам, если записи уже нет;
- создавать образ носителя и сканировать его;
- восстанавливать файлы с сохранением папок;
- показывать превью JPEG/PNG и EXIF;
- проверять найденные и восстановленные файлы;
- показывать шанс восстановления и причины;
- писать отчет и лог восстановления;
- работать без установки.

## Главное правило

Не восстанавливайте файлы на тот же носитель, с которого идет сканирование.

Если флешка важная или странно себя ведет, сначала сделайте образ, потом сканируйте образ.

## Сборка

```sh
make all
make test
make portable
```

Готовый архив появляется здесь:

```text
dist/RecuClassic-portable-win64.zip
```

## Ограничения

- NTFS пока нет.
- Глубокое сканирование может находить мусор.
- Если данные уже перезаписаны, файл может не открыться.
- Для чтения физических дисков Windows может попросить запуск от администратора.

## Нормальное использование

Восстанавливайте только свои данные или данные, на восстановление которых у вас есть разрешение.
